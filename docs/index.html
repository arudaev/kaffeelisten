HuggyWuggies // Kaffeelisten 06 / 11
Kaffeelisten
Monatsbericht
Was am Monatsende passiert

Cron um 22:00 UTC.
Niemand muss daran denken.

01 — Trigger

Vercel Cron feuert

Letzter Tag des Monats, 22:00 UTC. Oder Admin drückt manuell auf „Bericht senden".

Automatisch
02 — Aggregate

SQL gruppiert nach Firma → Person → Artikel

Postgres zählt Einträge, summiert Beträge, sortiert nach Firma.

~ 200 ms
03 — Render

PDF + Excel werden gebaut

Ein PDF pro Firma als Anhang. Excel-Workbook mit Pivot-fähiger Rohdatensicht.

~ 2 s
04 — Send

Resend liefert eine E-Mail aus

Empfänger: Geschäftsführer. Betreff: „Kaffeelisten — Mai 2026". Anhänge dabei.

Free Tier
05 — Reset

Live-Tabelle wird archiviert & geleert

Daten landen in transactions_archive. Neuer Monat startet bei Null.

3-Mon. rolling
